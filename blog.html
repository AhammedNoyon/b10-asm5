Important FAQ

What is the Document Object Model (DOM)?

represents and interacts with the structure of HTML and XML documents as a tree of objects.

How do you select an element from the DOM?

such as document.querySelector() to select the first matching element using a CSS selector, document.getElementById() to select an element by its unique ID, or document.getElementsByClassName() to select elements with a specific class name.

What is event delegation in the context of the DOM, and why is it useful?

Is added to a parent element instead of individual listeners for each child element. When an event occurs on a child element, it bubbles up to the parent, allowing the parent to handle the event. This approach reduces memory usage and improves performance, especially with dynamic content. It also simplifies code management by centralizing event handling in one place.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles using the DOM by accessing the element through methods like document.getElementById() or document.querySelector(). To change an attribute, use the setAttribute() method, such as element.setAttribute('src', 'image.jpg'), or directly modify properties, like element.src = 'image.jpg'. To alter styles, you can use the style property, such as element.style.color = 'red' to change the text color or element.style.width = '100px' to adjust the width.
